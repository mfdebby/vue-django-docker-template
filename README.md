## Prerequisites
1. [uv](https://docs.astral.sh/uv/)
2. [pnpm](https://pnpm.io/)
3. [Docker](https://www.docker.com/)

## Installation
```console
# clone repo
git git@github.com:freshstartagain/portfolio.git

# change directory
cd portfolio

# install dependencies
yarn install

# run dev server
yarn run dev
```

## Test
```console
# run test
yarn jest
```
